##Application Instructions
1. Run final_app_database_setup.py to initialize the database schema.
2. Run teams_players_full.py to fill with data to work with in nhlteams.db
3. Insert 1 as the user_id for all teams and players to be able to check logged in functionality
4. Run application.py to build the website and run the application.
5. Visit the application at http://localhost:5000
